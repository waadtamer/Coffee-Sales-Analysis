# Coffee-Sales-Analysis
Python data analysis on 2 years of coffee sales to uncover peak buying hours, top-selling coffee types, and weekly patterns. Includes data cleaning and visualization.
# ☕ Coffee Sales Analysis
## 📌 Project Overview
This project analyzes two years of coffee sales transaction data to uncover customer purchasing behavior and trends. By exploring variables like time of day, day of the week, and coffee types, the analysis provides data-driven insights into peak business hours and product popularity.
## 📊 Key Insights & Findings
 Peak Hours: Sales peak significantly in the morning before noon, making it the busiest period of the day.
 Top Sellers: Espresso  and  Latte emerge as the absolute best-selling coffee types.
 Weekly Patterns: Weekends show a slight drop in sales volume compared to weekdays, indicating strong demand from working professionals during the week.   Seasonality: Coffee remains an essential, stable part of customers' daily routines consistently throughout the entire year.
Tech Stack & Tools
Language: Python
Libraries: Pandas (Data Cleaning & Manipulation), NumPy, Matplotlib & Seaborn (Data Visualization)
Data Cleaning: Handling missing values, parsing datetime columns, and ensuring correct data types.
Exploratory Data Analysis (EDA): Grouping data by hours and days to extract trends.
Data Visualization:Creating charts to clearly represent the sales distribution over time and product categories.
